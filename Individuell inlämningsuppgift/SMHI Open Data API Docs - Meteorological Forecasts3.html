About

This demo is an example of using the API for getting a forecast for a point with javascript and JSON . There is no warranties for it's completeness but should be a good starting point for web developers. Have a look at the source of this page and the associated Javascript.

Try it

This demo itself does not do much. It is based on category pmp3g version 2. Select a point in the map. Data for the selected Parameter will show.
